# **Menú digital**
 ![Logo](https://drive.google.com/uc?export=view&id=13uhBE_dScPFVyRxmAqfek8Gq_NqaFzM6)

Este proyecto consiste en una carta virtual diseñada para restaurantes, bares y cafeterias. Pensada para ser accedida de forma rápida y práctica por los clientes a través de tablets del restobar. La aplicación web puede visualizar el menú desde cualquier diapositivo móvil vinculado a una red

## ***Características***
- Visualización del menú desde cualquier dispositivo móvil
- Acceso mediante tablet
- No requiere instalaciones ni apps externas
- Adaptable a cualquier tipo de establecimiento gastronómico
## Teconologias utilizadas 
**Frontend**
- Html
- Css
- Bootstrap
**Backend**
-Python if flask
-Mysql

# **Digital menu**

This project consists of a virtual menu designed for restaurants, bars and cafes. Designed to be accessed quickly and conveniently. The web application can visualize the menu from any mobile device linked to a network.

- Menu display from any mobile device
- No installation or external apps required
